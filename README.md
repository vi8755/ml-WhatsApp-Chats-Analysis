
# 📊 WhatsApp Chat Analysis (ML + Flask)

## 📌 Overview
This project is a web-based application that analyzes WhatsApp chat data and provides meaningful insights like user activity, message trends, and emoji usage.

---

## 🚀 Features
- 📈 Total messages, words, media, and links
- 👤 Most active users
- 🕒 Daily & monthly timeline analysis
- 🔥 Most common words
- 😂 Emoji analysis
- 🌐 Web interface using Flask

---

## 🧠 Tech Stack
- Python
- Flask
- Pandas
- NumPy
- Matplotlib / Seaborn

---

## 📂 Dataset
- WhatsApp exported chat file (.txt)
- No private or sensitive data included

---

 ## ⚙️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/whatsapp-chat-analysis-ml.git
```
2. Navigate to project folder:
 ```bash
 cd whatsapp-analyzer
```
3. Install dependencies:
```bash
pip install -r requirements.txt
```
4. Run the Flask app:
```bash
python app.py
```
5. Open in browser:
```bash
http://127.0.0.1:5000
```
